# Event frontend

## You can run App using the command

`npm run install`

`npm run dev`

## Use of technology

1. React
2. formik
3. fuse.js
4. react-intersection-observer
5. react-router-dom
6. react-select
7. react-chartjs-2
8. clsx
9. axios

## In this app you can

1. Get all events
2. Add participants for event
3. Get all participants from event
